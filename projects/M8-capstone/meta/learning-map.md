## Opções de Projeto

Escolha 1 na Semana 71:

| Projeto | Escopo | Linguagem sugerida |
|---------|--------|--------------------|
| **Database** | Storage engine + queries (SELECT/INSERT/DELETE) sem transactions complexas | Zig ou Go |
| **Distributed KV Store** | KV store com replicação básica (sem Raft completo) | Go |
| **Language** | Interpreter M7 + features: loops, closures, built-ins | Zig |
| **HTTP Framework** | Router + middleware + handler pipeline | Go |

## Benchmark Q4 (Semana 84)

**Critério de sucesso**: Demo funcional + documentação de arquitetura.

Checklist:
- [ ] Projeto funcionando com demo ao vivo (sem script)
- [ ] Arquitetura explicada em 10 minutos
- [ ] Decisões de trade-off justificadas
- [ ] Perguntas técnicas respondidas sobre qualquer componente
- [ ] README legível por outro engenheiro
