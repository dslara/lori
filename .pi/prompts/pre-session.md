# Pré-Sessão Lori

Antes de começar, execute o ritual de preparação:

1. **Objetivo claro**: "Nesta sessão vou _______"
2. **Técnica escolhida**: pomodoro / feynman / drill / retrieval / srs / directness
3. **Timer**: /lori-timer start [minutos]
4. **Proteção**: silencie notificações externas, feche abas irrelevantes
5. **Material**: tenha à mão apenas recursos do módulo ativo

> Regra: sem objetivo claro, não há sessão.
